# Classes

Свой файл с описанием полученного класса называем **строго** по паттерну `ist0?_<fio>_<class>`. За промах с названиеи будут снижены баллы. Что бы доказать свое присутствие на паре впишите свою фамилию в список ниже в течение временного интервала проведения пары. Что бы замержить изменения надо сделать ветку по паттерну `ist_0?/<fio>`.
